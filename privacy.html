Privacy Policy

Idle Realm does not collect, store, or transmit any personal data.

This application runs entirely in the browser. There is no backend server and no external database.

All game progress and settings are saved locally on your device using localStorage. Clearing your browser data will permanently remove your progress.

The Pi SDK, if present, is used only for authentication and user identification within the Pi Network ecosystem. No personal information is accessed or stored by Idle Realm.

Idle Realm does not use analytics, cookies, trackers, or third-party services.

This page is fully readable without JavaScript enabled.
